# Qaert

Qaert is a QA Telegram Bot based on BERT model.

## Installation & Contributing

Use the [DEVELOPER.md](./DEVELOPER.md) guide to run or contribute to the project.

## Usage

```python
python bot.py
```

or using Docker:

```bash
docker build --tag qaert .
docker run -d qaert
```
